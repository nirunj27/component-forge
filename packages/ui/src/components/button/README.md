# Button

Generic configurable button

> Generated by [Component Forge](https://github.com) — edit the schema and re-run `forge generate`.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `primary \| secondary \| ghost \| danger` | primary | Visual style |
| `size` | `sm \| md \| jjjjjjj` | md | — |
| `disabled` | `boolean` | false | — |
| `loading` | `boolean` | false | — |

## Accessibility

- Role: `button`
- Keyboard: Enter, Space


## Usage

```tsx
import { Button } from "@component-forge/ui/button";

<Button>Content</${schema.name}>
```
